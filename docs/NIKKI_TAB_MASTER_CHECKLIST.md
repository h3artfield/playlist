# Nikki workbook — master tab checklist

**Workbook:** `2024 Nikki Spreadsheets.xlsx` (see `nikki_workbook` in `config/april_2026.yaml`).

**Tab count:** 48

## How to use this

1. Work **top to bottom** (same order as Excel tabs / audit doc).
2. For each tab, open **[NIKKI_WORKBOOK_TAB_AUDIT.md](./NIKKI_WORKBOOK_TAB_AUDIT.md)** for a **preview** + **try-load episode count** (uses the same `nikki.load_sheet` path as the app; YAML `nikki_row_filter` applies when that tab is wired in `april_2026.yaml`).
3. When parser style, optional `nikki_columns`, and any **NOTE** / green / special rules are correct for that tab, check the box here.
4. After large Excel edits, re-run: `python scripts/audit_nikki_workbook.py`

**Start here:** [NIKKI_WORKBOOK_TAB_AUDIT.md](./NIKKI_WORKBOOK_TAB_AUDIT.md) opens with a **priority table** (tabs that currently parse **0** episodes and need column/parser work).
